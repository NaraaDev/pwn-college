# The root

> Module: **Linux Luminarium / Pondering Paths** · Challenge: `the-root`

Linux файлын систем нь **`/`** (root) гэдэг ганц цэгээс эхэлдэг модны бүтэц. Бүх зам энэ цэгээс эхэлж "ургадаг".

## Тайлбар

- `/` — **root директор**, файлын модны хамгийн дээд цэг.
- `cd /` — root руу шилжих.
- `./pwn` — одоогийн (`.`) директорт байгаа `pwn` програмыг ажиллуулах. Энэ challenge-д root доорх `pwn`-ыг дуудаж байна.
