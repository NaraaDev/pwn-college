# Comparing Files

> Module: **Comprehending Commands** · Challenge: `comparing-files`

> 📝 *Бодолгүй / тэмдэглэлгүй үлдсэн challenge.*

Хоёр файлыг хооронд нь харьцуулах коммандууд.

## Тайлбар

- `diff <a> <b>` — мөр мөрөөр харьцуулна. Ялгаа байвал доорх форматаар хэвлэнэ:
  - `<` — `a`-д л байгаа
  - `>` — `b`-д л байгаа
- `cmp <a> <b>` — байт-аар харьцуулна. Анх ялгаа гарсан байрлалыг хэвлэнэ.
- `md5sum <file>` — файлын hash. Хоёр файлын hash адил бол агуулга нь адил.

```bash
diff a.txt b.txt
diff -u a.txt b.txt          # unified формат, patch болгоход тохиромжтой
md5sum a.txt b.txt           # хурдан тэнцүү эсэх шалгах
```
