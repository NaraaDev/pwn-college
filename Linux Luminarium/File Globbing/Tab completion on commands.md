# Tab Completion on Commands

> Module: **Linux Luminarium / File Globbing** · Challenge: `tab-completion-on-commands`

Tab autocompletion зөвхөн файл, замд төдийгүй ажиллуулах командын нэрэнд ч ажилладгийг ашиглана.

## Тайлбар

- `/challenge/bin/` дотор санамсаргүй тоон дагавартай (`pwncollege-<random>`) ганц ажиллах файл байрладаг тул нэрийг бүхэлд нь мэдэхгүй ч Tab дарж autocomplete хийснээр бодит нэрийг олж, ажиллуулна.
- Тухайн программыг ажиллуулахад шууд флаг хэвлэдэг.

## Solution

```console
hacker@globbing~tab-completion-on-commands:~$ /challenge/bin/pwncollege-1882
Correct! Here is your flag:
pwn.college{ozapVyCfiVbmqXUsR105qEqpNOF.0VN0EzNxwyMwEzM2EzW}
```
