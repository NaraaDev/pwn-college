## Cheatsheet

| Комманд | Зорилго | Жишээ |
|---------|---------|-------|
| `cat <file>` | Файлын агуулга хэвлэх | `cat /flag` |
| `ls [dir]` | Директор жагсаах | `ls /challenge` |
| `grep <pattern> <file>` | Хэв олох | `grep error log.txt` |
| `touch <file>` | Хоосон файл үүсгэх / time нэмэх | `touch pwn` |
| `cp <src> <dst>` | Хуулах | `cp /flag /tmp/x` |
| `mv <src> <dst>` | Зөөх / нэр өөрчлөх | `mv old new` |
| `rm <file>` | Устгах | `rm delete_me` |
| `diff <a> <b>` | Файл харьцуулах | `diff a.txt b.txt` |
