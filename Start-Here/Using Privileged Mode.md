# Using Privileged Mode

> Module: **Start Here** · Challenge: `using-privileged-mode`

pwn.college challenge-уудыг root эрхтэй (privileged) горимд эхлүүлж болохыг харуулна. Зарим даалгаварт root байх шаардлагатай.

## Тайлбар

- **Privileged mode** — challenge орчин root хэрэглэгчээр эхлэх горим. UI дээр *"Start (Privileged)"* товч.
- **`sudo`** — Linux-д энгийн хэрэглэгчээс root түр шилжих комманд. Privileged горимд `sudo` шаардлагагүй, аль хэдийн root байна.
- **`whoami`** — одоо ямар хэрэглэгчээр нэвтэрсэнг харуулна.

```bash
whoami   # root гэж буцаавал privileged горим
```
