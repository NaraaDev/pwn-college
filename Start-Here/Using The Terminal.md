# Using The Terminal

> Module: **Start Here** · Challenge: `using-the-terminal`

Terminal (нэрлэдэг "shell" эсвэл "command line") юу болохыг ойлгох. Bash-ын `$` prompt дээр комманд бичээд `Enter` дарна.

## Тайлбар

- **Terminal** — текст хэлбэрээр компьютертой харилцах үндсэн хэрэгсэл.
- **Shell** — терминал доторх програм. pwn.college дээр `bash` ашигладаг.
- **Prompt** — `hacker@hostname:~$` гэх мэт мөр. `~` нь home директор (`/home/hacker`), `$` нь оролт хүлээж буй тэмдэг.
