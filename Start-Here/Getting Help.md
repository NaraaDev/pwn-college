# Getting Help

> Module: **Start Here** · Challenge: `getting-help`

pwn.college болон Linux дахь тусламжийн эх үүсвэрүүдтэй танилцана.

## Тайлбар

- **Discord** — pwn.college-ийн оюутны discord сервер. Асуудал тулгарвал тэнд асуу.
- **`man <command>`** — Linux-ийн manual page. Ж.нь `man ls` — `ls` коммандын тайлбар, флаг, жишээ.
- **`<command> --help`** — олон комманд богино help мэдээлэл өгдөг.
- **`apropos <keyword>`** — түлхүүр үгээр man хуудас хайх.

```bash
man ls              # ls коммандын гарын авлага
ls --help           # ls-ийн товч тусламж
apropos copy        # "copy" агуулсан коммандуудыг олох
```
